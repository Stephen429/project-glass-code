# LDVTS-SSLG Project GLASS
Project GLASS is an independent, student-led transparency initiative developed by the **Supreme Secondary Learner Government** *(SSLG)* of **Lydia D. Villangca Trade School**.

Designed to serve as a central digital gateway, the project aims to foster institutional accountability by streamlining access to official school records, policies, and ongoing student-led initiatives. By providing a responsive, user-friendly dashboard, Project GLASS empowers the student body with the information they need, ensuring that leadership remains open, accessible, and grounded in service.

### Key Features

* **Centralized Records:** Easy access to documents, gazettes, and government updates.
* **Transparent Governance:** Public-facing dashboard for tracking accountability.
* **Community Engagement:** Integrated feedback channels and active project tracking.
* **Responsive Design:** Fully optimized for mobile and desktop viewing to support the entire campus community.
